
PROYECTO FINAL BUNKER4

1- Crear una carpeta de project y luego las carperpetas de Front y Back dentro.
	$mkdir project && cd project
	$mkdir frontend && mkdir backend  && cd backend
	$ code .
	
2- Avtivar el ambiente virtual de py para que las dependencias se instalen solamente aca.

	$python3 -m venv venv 
	# para que se instalen todas las dependencias solo en este proyecto.
	
	$source venv/bin/activate
	# para activar el entorno virtual. Se tiene que ver (venc)

3- Instala las dependencias necesarias. Asegurarse de estar en venv
	venv)$pip install fastapi uvicorn sqlalchemy pymysql

	SQLAlchemy es una librería de Python que facilita la interacción con bases de datos relacionales. 
Permite a los desarrolladores trabajar con bases de datos mediante objetos Python, en lugar de escribir consultas SQL directamente. 
También ofrece herramientas para la gestión de tablas y la creación de consultas complejas. 

	PyMySQL es una biblioteca de Python para interactuar con bases de datos MySQL, escrita completamente en Python.

4- Crear el requierements.txt con todas las dependencias instaladas para futuras usos.

	$pip freeze > requirements.txt 

5- Iniciar GIT

	$git init
	#crear .gitignore con www.toptal.com  (crea automaticamente el ignore a nuestras necesidades.

6- Ejecutar el servidor web

	$$uvicorn main:app --reload
